# Task list bot

### Features:
 - Python
 - aiogram 3.20.0
 - aiogram-dialog 2.3.1
 - MongoDB (beanie)

### Install
```bash
git clone https://github.com/sqlmerr/tasklistbot
cd tasklistbot

# install uv (https://docs.astral.sh/uv/getting-started/installation/#installation-methods)
curl -LsSf https://astral.sh/uv/install.sh | sh 
```

### Run
```bash
uv run python -m bot
```
